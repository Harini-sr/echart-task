Static Chart

I have created a static EChart with hardcoded data.

Dynamic Chart

A Dynamic Chart updates its data or type based on user interaction or real-time input. This helps make data more interactive and personalized for the viewer.

Select Option Chart

A Select Option Chart allows users to choose different chart types (like line, bar, pie, or doughnut) using a dropdown. This enhances flexibility and user control in data visualization.
